# Vue3电商项目
> **前言:**
>
> 此项目是基于Vue3的电商网站类项目
>
> 技术栈: Vue3 + Vuex + Vue-router + Axios
>
> 后端接口可能会失效

>运行环境
>
>克隆源码➡package.json所属文件下npm i下载依赖，之后npm run dev运行，node18以上运行npm run dev_t
>
>在 [http://localhost:8080](http://localhost:8888/) 访问项目
